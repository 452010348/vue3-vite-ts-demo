# spl-token 命令行工具列表
spl-token <命令> --help
```sh
spl-token accounts                                列出所有拥有者的令牌账户
spl-token address                                 获取钱包地址
spl-token apply-pending-balance                   将待处理的加密令牌从待处理余额收集到可用余额
spl-token approve                                 为令牌账户批准委托人
spl-token authorize                               授权新的签名密钥对于令牌或令牌账户
spl-token balance                                 获取令牌账户余额
spl-token bench                                   令牌基准测试工具
spl-token burn                                    从账户销毁令牌
spl-token close                                   关闭令牌账户
spl-token close-mint                              关闭令牌铸币机
spl-token configure-confidential-transfer-account 配置令牌账户的机密转账
spl-token create-account                          创建新的令牌账户
spl-token create-multisig                         创建描述M:N多签名的新账户
spl-token create-token                            创建新的令牌
spl-token deposit-confidential-tokens             存入机密转账的金额
spl-token disable-confidential-credits            禁用令牌账户的机密转账
spl-token disable-cpi-guard                       禁用令牌账户的CPI保护
spl-token disable-non-confidential-credits        禁用令牌账户的非机密转账
spl-token disable-required-transfer-memos         禁用令牌账户的必需转账备注
spl-token display                                 根据地址查询SPL令牌铸币机、账户或多签名的详细信息
spl-token enable-confidential-credits             启用令牌账户的机密转账。首次启用机密转账，请使用`configure-confidential-transfer-account`命令。
spl-token enable-cpi-guard                        启用令牌账户的CPI保护
spl-token enable-non-confidential-credits         启用令牌账户的非机密转账
spl-token enable-required-transfer-memos          启用令牌账户的必需转账备注
spl-token freeze                                  冻结令牌账户
spl-token gc                                      清理不必要的令牌账户
spl-token help                                    打印此消息或给定子命令的帮助信息
spl-token initialize-metadata                     初始化令牌铸币机的元数据扩展
spl-token mint                                    铸造新的令牌
spl-token revoke                                  撤销委托人的权限
spl-token set-interest-rate                       设置利率给有利息的令牌
spl-token set-transfer-fee                        对已配置转账费用的令牌设置转账费用
spl-token set-transfer-hook                       为令牌设置转账钩子程序ID
spl-token supply                                  获取令牌供应量
spl-token sync-native                             将本地SOL令牌账户与其底层的lamports同步
spl-token thaw                                    解冻令牌账户
spl-token transfer                                在账户之间转移令牌
spl-token unwrap                                  将SOL令牌账户解包
spl-token update-confidential-transfer-settings   更新令牌的机密转账配置
spl-token update-default-account-state            更新铸币机的默认账户状态。需要默认账户状态扩展。
spl-token update-group-address                    更新铸币机的群组指针地址。需要群组指针扩展。
spl-token update-member-address                   更新铸币机的群组成员指针地址。需要群组成员指针扩展。
spl-token update-metadata                         更新带有元数据扩展的令牌铸币机的元数据
spl-token update-metadata-address                 更新铸币机的元数据指针地址。需要元数据指针扩展。
spl-token withdraw-confidential-tokens            提取机密转账的金额
spl-token withdraw-excess-lamports                从令牌程序拥有的账户中提取lamports
spl-token withdraw-withheld-tokens                从铸币机和/或账户中提取被扣留的转账费用令牌
spl-token wrap                                    将本地SOL包装成SOL令牌账户
```
